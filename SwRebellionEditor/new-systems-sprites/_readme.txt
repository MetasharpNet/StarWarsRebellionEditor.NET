Place in this folder files that will be imported as
sprites in strategy.dll bitmap resources.

name is split on '-' character to determine the
bitmap resource entry.

Input pictures must be in 37x37 with a pure blue background.
Inside this square we advise a centered:
- 37x37 sprite for giant gas
- 35x35 sprite for planet
- 31x31 sprite for moon

Example:
14001-abregado-rae.bmp => 14001
14002-agamar.bmp       => 14002
...
