Place in this folder files that will be imported as
resources in tactical.dll 303 resources folder.

name is split on '-' character to determine the
resource entry in 303 folder. +1000 is added for palette

Input pictures must be in 256x256 with a pure black background and in 8bits palette indexed colors or in 256 colors directly.

Example:
14001-abregado-rae.png => 14001 for bin image, 15001 for bin palette
14002-agamar.png       => 14002 for bin image, 15002 for bin palette
...
