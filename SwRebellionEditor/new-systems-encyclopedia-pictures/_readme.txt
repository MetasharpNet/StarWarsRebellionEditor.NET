Place in this folder files that will be imported as
EDATA pictures in the EDATA folder.

name is split on '-' character to determine the
bitmap resource entry.

Input pictures must be in 400x200 with a stars background.
Inside this rectangle we advise a centered:
- 180x180 sprite for giant gas
- 150x150 sprite for planet
- 120x120 sprite for moon

Example:
14001-abregado-rae.bmp => EDATA.14001
14002-agamar.bmp       => EDATA.14002
...
